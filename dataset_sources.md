# 📊 Dataset Sources

## Supervised Learning

### Classification
1. **Iris Dataset**
   - Source: UCI Machine Learning Repository
   - URL: https://archive.ics.uci.edu/ml/datasets/iris
   - Description: 150 samples, 4 features, 3 classes

2. **Pima Indians Diabetes Database**
   - Source: UCI Machine Learning Repository
   - URL: https://www.kaggle.com/uciml/pima-indians-diabetes-database
   - Description: 768 samples, 8 features, binary classification

### Regression
3. **California Housing Dataset**
   - Source: Scikit-learn built-in dataset
   - Description: 20,640 samples, 8 features, continuous target

## Unsupervised Learning

### Clustering
4. **Mall Customers Dataset**
   - Source: Kaggle
   - URL: https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python
   - Description: 200 samples, customer income and spending score
