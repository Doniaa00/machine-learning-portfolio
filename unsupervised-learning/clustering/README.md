# 🔍 Clustering Projects

This folder contains clustering projects for unsupervised learning.

## Projects

### 1. Customer Segmentation (`01_customer_segmentation.ipynb`)
- **Goal**: Segment customers based on income and spending behavior
- **Dataset**: Mall Customers (200 samples)
- **Algorithm**: K-Means Clustering
- **Key Concepts**: Elbow method, silhouette score, cluster interpretation

## Key Learnings
- K-Means effectively groups similar customers
- Optimal cluster selection requires domain knowledge
- Clustering enables targeted marketing strategies
