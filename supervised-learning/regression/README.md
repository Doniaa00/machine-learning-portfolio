# 📈 Regression Projects

This folder contains regression projects predicting continuous values.

## Projects

### 1. California Housing - Linear Regression (`01_california_housing_linear.ipynb`)
- **Goal**: Predict median house values in California
- **Dataset**: California Housing (20,640 samples, 8 features)
- **Algorithm**: Linear Regression
- **Key Concepts**: Feature coefficients, residual analysis, R² score

### 2. California Housing - Random Forest (`02_california_housing_random_forest.ipynb`)
- **Goal**: Compare Random Forest vs Decision Tree performance
- **Dataset**: California Housing
- **Algorithm**: Random Forest Regressor
- **Key Concepts**: Ensemble methods, feature importance, hyperparameter tuning

## Key Learnings
- Random Forest reduces MSE by ~50% compared to single trees
- Feature importance helps identify key predictors
- Ensemble methods reduce overfitting
