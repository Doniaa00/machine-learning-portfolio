# 🎯 Classification Projects

This folder contains classification projects demonstrating various algorithms and techniques.

## Projects

### 1. Iris Flower Classification (`01_iris_classification.ipynb`)
- **Goal**: Classify iris species (Setosa, Versicolor, Virginica)
- **Dataset**: Iris (150 samples, 4 features)
- **Algorithm**: Logistic Regression
- **Key Concepts**: Multi-class classification, feature scaling, decision boundaries

### 2. Diabetes Diagnosis (`02_diabetes_diagnosis.ipynb`)
- **Goal**: Predict diabetes presence from medical measurements
- **Dataset**: Pima Indians Diabetes (768 samples, 8 features)
- **Algorithm**: Logistic Regression
- **Key Concepts**: Medical diagnosis, recall optimization, ethical AI

## Key Learnings
- Logistic Regression provides interpretable results
- Class imbalance handling
- Medical applications require careful metric selection
